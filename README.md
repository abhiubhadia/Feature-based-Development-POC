# Feature Based development
This is Feature Based Development buitl using Polymer Web components and node express to replace Standard Monolithic UI.
#Polymer #Angular #Express/Node

Steps to run application
```
git clone https://github.com/abhiubhadia/Feature-based-Development-POC.git

make sure you have node,mongoD and bower installed.

npm install
bower install (optional if it doesn't work through npm install)
```

Go to /public/javascripts directory
```
npm install ( I know it sounds wierd but I have used angula2 alpha version through 
npm and for dependency security it is not included in main node modules)
```

Go to Mongo Data Directory
```
copy both files from project /data folder and paste to local mongo /data directory
```

Go to Browser and hit
```
http://localhost:3000
```
